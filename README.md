# MediAI – AI-Based Healthcare Symptom Prediction System Using Machine Learning

## Project Description

MediAI is an AI-Based Healthcare Symptom Prediction System developed using Machine Learning concepts and web technologies. The system predicts diseases based on symptoms entered by users and provides healthcare recommendations and precautions.

The project demonstrates the practical implementation of Artificial Intelligence and Data Science technologies in the healthcare sector. The application is designed with a modern responsive interface and includes healthcare analytics, dashboard reports, symptom analysis, and disease prediction features.

This project was developed as part of the PGDM AI & DS final academic project.

---

# Project Objectives

- To develop an AI-based healthcare prediction system
- To predict diseases using symptoms entered by users
